# help_girlfriends


help_girlfriends 是一款基于微信公众号和推送通知的应用，旨在帮助女朋友解决丢三落四的毛病。通过定时提醒女朋友，帮助她养成良好的生活习惯。

## 如何使用

1. 在微信中搜索并关注 Help_Girlfriends 公众号。
2. 将女朋友的push deer添加到已注册的用户列表中。
3. 在设置页面中，选择提醒时间和频率。
4. 等待每天的定时提醒，帮助女朋友养成好习惯。
